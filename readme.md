## position
    static : 기본흐름에 맞춰서 
    relative : 기본흐름에 따라 배치, 좌표 속성 사용 가능
        --relative-top
        --relative-bottom
        --relative-right
        --relative-left
    absoulute : 절대적인 좌표위치에 따라 배치
    fixed : 스크롤해도 해당 위치에 고정 (절대적 좌표위치)
    sticky : static처럼 기본 흐름에 따라 배치, 지정한 좌표의 임계점에 이르면 fixed
